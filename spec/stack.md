# Stack: TypeScript + React + Vite

**Maturity**: Alpha - Velocity over stability. Breaking changes expected.

> **Package Manager**: npm
> **Test Runner**: Vitest
> **Build Tool**: Vite

## Lint
```bash
npm run lint --fix
```

## Typecheck
```bash
npm run typecheck
```

## Test
```bash
npm test
```

## Build
```bash
npm run build
```

## E2E Tests (Optional)
```bash
# If playwright.config.ts exists
npm run test:e2e
```
